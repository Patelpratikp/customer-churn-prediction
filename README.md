# Customer Churn Prediction & Risk Segmentation

## Project Objective
Predict customer churn using Machine Learning and segment customers into risk categories for proactive retention strategies.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Plotly

## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting

## Key Insights
- Month-to-month customers had highest churn (42.7%)
- Low-tenure customers were more likely to churn
- Higher monthly charges increased churn probability

## Business Recommendations
- Offer long-term contract discounts
- Create onboarding retention programs
